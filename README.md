NimblyChallenge (or Nibucha)
---

### Introduction
Developed with React and Redux, the projects creates a simple UI for weather querying.

### Usage

#### install
```sh
$ git clone https://github.com/shuchenliu/NimblyChallenge.git
$ cd NimblyChallenge
$ npm install
```

#### Run & test

```sh
$ npm start
```

#### Build

```sh
$ npm run build
```
